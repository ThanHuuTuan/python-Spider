#### 编码问题
参考文章：https://www.cnblogs.com/jdy113/p/8034771.html。<br>
因为python2和3的系统编码是有区别的，2的是ASCII，而3的是UTF-8。所以看博主的代码是需要添加编码转换代码的，但是即便是utf-8，上一篇文章在爬取qq空间的时候发现有部分颜文字是无法写入到本地的txt文本的又该如何解决呢？
```python
import sys
reload(sys)
sys.setdefaultencoding('utf-8')
```
